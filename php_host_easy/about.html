About the Procurement System

A purpose-built internal tool for Magenta Investments LLC to manage purchase requests, vendor relationships, and department tenders — all in one secure platform.

✨ Key Features
📋

Purchase Requests

Staff submit purchase requests with budget estimates, priority levels, and supporting attachments for manager review.

✅

Approval Workflows

Requests route through manager → department head → CEO based on the requested amount and configured approval thresholds.

📢

Department Tenders

Departments publish procurement tenders. Access is role-controlled — admins decide which staff members can view and respond.

🏪

Vendor Portal

Vendors self-register, log in, view open tenders matching their supply categories, and submit competitive quotes.

📁

File Attachments

Purchase requests and vendor registrations support secure document uploads (quotations, specs, company profiles).

📧

Email Notifications

Automatic email alerts on request creation, approval, and rejection via Microsoft 365 SMTP — keeping everyone informed.

👥

User Management

Admin panel to create staff accounts, assign departments and roles, activate/deactivate users, and manage permissions.

🔐

Secure by Design

JWT authentication, bcrypt password hashing, rate limiting on login, and security headers on every response.

👤 User Roles
⚙️

Admin

Full system access — manages users, settings, and all data.

👑

CEO

Final approver for high-value requests. Views all company data.

🏢

Department Head

Approves requests from their department up to the threshold.

👔

Manager

First-level approver for employee purchase requests.

🧑‍💼

Employee

Submits purchase requests and tracks approval status.

🏪

Vendor

External supplier — accesses the vendor portal only.

🛠️ Technology

The system comes in two flavours: a modern Next.js build (recommended for VPS/cloud) and a lightweight PHP + vanilla JS build for shared hosting environments — both backed by MongoDB Atlas.

⚛️ Next.js 14 🐘 PHP 8.1 🍃 MongoDB 🔑 JWT Auth 🔒 bcrypt 📧 SMTP / Office 365 🎨 Tailwind CSS 🌐 Vanilla JS
